Child Selectors

The child selector selects all elements that are the children of a specified element.
In this example all h2 elements which are children of div elements get the color orange.
Syntax used "div > h2"


Adjacent selectors

The adjacent sibling selector selects all elements that are
the adjacent siblings of a specified element. Let us try this. First we create an element.
I choose a section element.

I am inside a section element. Now we place another element directly AFTER it

I am a h4 element directly placed after the section element. We target it like this:
"section + h4" and the color property is given another value....


General selectors

The general sibling selector selects all elements that are siblings of a specified element.
Example time: Let us try and target all paragraph elements which are siblings of h3 elements! Et Voila!
Syntax: "h3 ~ p"

Last one. Cascading and inheritance. Laten we dit uitleggen in mijn moedertaal.

Inheritance, of overerving, is het proces waarbij bepaalde eigenschappen (je raadt het al)
worden overgeërfd. Het lettertype is een voorbeeld van een eigenschap die ouders meegeven aan hun kinderen.
Waarom? Omdat het onhandig is als je dit voor elke sectie apart moet instellen.

Achtergrondkleur is een eigenschap die niet automatisch wordt doorgegeven.
Achtergrondkleur "verdwijnt" en volgende generaties zullen
deze eigenschap zelf moeten ontdekken. Vanaf de geboorte
bevinden ze zich in een doorzichtige, kleurloze omgeving en wie van hen in
in staat is hieraan te ontsnappen, wordt beslecht in een hevige strijd
die slechts door een enkeling aanschouwen wordt.
Waarom? Omdat het eigenschappen zijn die je meestal eenmalig wilt gebruiken.

Cascade gaat over wat prioriteit krijgt in het geval van een conflict. Er zijn drie regels:
1. Eigenschappen die later zijn ontstaan vervangen eerder verworven eigenschappen.
2. Specifiekere selectors krijgen meer prioriteit.
3. Eigenschappen die je zelf instelt krijgen een hogere prioriteit dan eigenschappen die zijn overgeërfd.
