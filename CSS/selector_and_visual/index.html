Three ways to load CSS

The first one. Plonked straight into HTML! This is done using the style tag.
The second one is inside the head element. Styling is applied to the entire page.
Final one is external. A seperate CSS file. Every page can have its own css file.

CSS Syntax

W3 schools states the following: It consists of a selector and a declaration block! What does this mean?
The selector is the element you want to target, for example h1, li, or section.
The declaration part consists of properties and values. Color is a value and a value could be yellow, or even brown.
Sooooo, values are assigned to properties.

Types of selectors

Type, descendant and class selectors. Let us give an example for each! First type selector:
This selects the element with the corresponding node name, like p, or h5 tags.
Next one is a descendant selector. This one matches ALL elements which are descendants of a specified element.

Last one class selector. You give an element, of your choosing, a classname and target it with CSS.
